This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

Task is to build a web application based on the provided Figma design

## Implementation

- Using Framework [Next.js](https://nextjs.org) & TypeScript
  - Reusable Components
  - Zustand for global state management
- Styling: Tailwindcss CSS v.4
- Animations: use GSAP for ReactJS & SwiperJS 

## Deployment
Code Repository: [https://github.com/Imanullah/jc](https://github.com/Imanullah/jc)
